# Terminal TicTacToe

![preview](./preview.png)

## How to start

```console
$ make
$ ./ttt
```

## Controls

| key                                                 | description        |
|-----------------------------------------------------|--------------------|
| <kbd>w</kbd>,<kbd>a</kbd>,<kbd>s</kbd>,<kbd>d</kbd> | Move cursor        |
| <kbd>SPACE</kbd>                                    | Select Cell        |
| <kbd>r</kbd>                                        | Restart            |
| <kbd>q</kbd>                                        | Quit               |
